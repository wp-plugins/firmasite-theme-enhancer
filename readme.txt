== Description ==

This plugin provides new features to themes. Twitter Bootstrap design elements, custom editor buttons and more. 

This plugin is supporting FirmaSite theme's theme style changes. I suggest you use theme + plugin together.
You can find detailed information, showcase, live demo, tips and tricks about theme in: http://theme.firmasite.com/
You can check screenshots or watch this short video:
http://www.youtube.com/watch?v=TB24eowl_2E

Sponsored by: [FirmaSite](http://firmasite.com/)
